Why Do You Need a Strong Password?

Weak passwords are the leading cause of account hacking. A password like "password123" can be cracked in under a second using modern brute-force tools. Cybercriminals use automated software that tests billions of password combinations per second. A truly strong password — long, random and unique — is your first line of defence against unauthorised access.

Password Security Best Practices

  • Use a unique password for every account — if one site is breached, others stay safe
  • Use a password manager — tools like Bitwarden, 1Password or Google Password Manager store and auto-fill passwords securely
  • Enable two-factor authentication (2FA) — even if your password is stolen, 2FA blocks access
  • Never share passwords — not even with IT support (legitimate teams never ask for your password)
  • Change passwords after breaches — check haveibeenpwned.com to see if your email has appeared in data leaks
  • Avoid common substitutions — "P@ssw0rd" is still weak — attackers know these patterns

Frequently Asked Questions

How long should my password be?At minimum 12 characters for general accounts. For sensitive accounts (banking, email, social media), use 16+ characters. Length is the single most important factor in password strength.
Are generated passwords stored anywhere?No. Passwords are generated entirely in your browser using JavaScript. They are never transmitted to any server and are not saved anywhere.
What special characters should I include?Common special characters are ! @ # $ % ^ & * ( ) – _ = + [ ] { } ; : , . ? /. Some websites restrict certain special characters — if a generated password is rejected, try regenerating without the symbol that caused the issue.
Should I write down my password?Avoid writing passwords on sticky notes or in plaintext files. Use a reputable password manager instead. If you must write a password down (e.g. for a master account), store it in a physically secure location.
